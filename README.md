# VR Flappy Wings

A simple VR game inspired by Flappy Bird, built with React and Three.js.

## How to Play
- The player is stationary; the pipes move toward you.
- Gravity pulls you down continuously.
- Flap by lifting the controllers above shoulder level and pushing them down.
- Avoid hitting the pipes and stay within the vertical limits.

## Controls
- VR: Move the controllers up and down to flap.
- Desktop: Press Space to flap.
- Restart the game by pressing any controller button after game over.

## Tech Stack
- React
- @react-three/fiber
- Three.js
- WebXR
- HDRI environment lighting
